# selenium-java-framework

<h3>Framework for integration testing of Juice-Shop application</h3> <br/>

Run : mvn clean test <br/>
Generate report: mvn allure:serve <br/>

Language: Java 8  <br/>
Running: TestNG  <br/>
Logging: Logback <br/>
Reports: Allure <br/>
Assertion: Hamcrest <br/>
JSON parser: GSON <br/>
REST: Rest Assured <br/>
